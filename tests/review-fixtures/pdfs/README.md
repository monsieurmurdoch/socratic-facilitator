# PDF Fixtures To Add

Drop sample PDFs for human review into this folder.

Recommended set:

1. `born-digital-poem.pdf`
   A clean PDF with selectable text and obvious line breaks.

2. `clean-scan-poem.pdf`
   A scanned poem or excerpt that is image-based but still readable.

3. `rough-scan-handout.pdf`
   A lower-quality classroom handout with skew, blur, or uneven contrast.

Optional companion files:

- `born-digital-poem.expected.txt`
- `clean-scan-poem.expected.txt`
- `rough-scan-handout.expected.txt`

If an `.expected.txt` file exists beside a PDF, the review script will include a simple similarity comparison in the report.
